## Keyboard Shortcuts (MacOS)

* `cmd+shift+tab` -- opposite of `cmd+tab`
* `cmd+shift+p` -- while in chrome devtools, open chrome devtools command pallete
* `cmd+alt+i` -- open chrome devtools
